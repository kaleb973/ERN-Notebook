# ERN: Electronic Research Notebook 

An open-source, web-first Electronic Science Notebook tailored for students, hackers, and R&D enthusiasts (especially in Biomedical Engineering). Built to replace rigid, bloated legacy tools with a fast, zero-friction canvas for logging experiments, tracking variables, and visualizing data.

> **Status:** 🛠️ Work In Progress — Building live for Hack Club Stardance!
> **Live Demo:** https://ern-notebook.vercel.app/

---

## The Vision
Most scientific logging tools are slow, expensive, or overly corporate. ERN provides a fluid, dark-mode desktop canvas optimized for speed at the lab bench:
* **Inline WYSIWYG Canvas:** Type notes, use instant markdown shortcuts, and drop in elements on a single clean sheet.
* **Hardware & Data Agnostic:** Built to make dropping in raw CSV files or hooking up microcontrollers (like Arduino/ESP32 via web serial) as simple as adding a modular plugin.
* **Frictionless Interface:** A global `Ctrl/Cmd+K` command palette allows lightning-fast navigation, project switching, and quick capture logging.

---

## Tech Stack
* **Framework:** Next.js (React + TypeScript)
* **Styling:** Tailwind CSS (Dark Mode Optimization)
* **Text Engine:** TipTap (Headless WYSIWYG Framework powered by ProseMirror)
* **UI Components & Icons:** Lucide React & `ctrl/cmd+k`
